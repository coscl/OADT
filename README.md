openstack_deploy
================

openstack deploy project for COSCL

Steps

1 安装部署源
    Install OS : NeoKylin Server 6.0（在安装过程中选择虚拟化组件）
    mkdir /opt/openstack/
    cd /opt/openstack
    Download the Source code
    
2 执行安装脚本
    sh install_ns6.0.sh

3 登录web界面
    server_ip:8000/index
